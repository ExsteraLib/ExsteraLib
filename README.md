# ExsteraLib

**ExsteraLib** — это SDK (Software Development Kit) для разработки плагинов **ExsteraGram**. Библиотека предоставляет единый высокоуровневый API для работы с возможностями клиента и значительно упрощает создание плагинов, избавляя разработчиков от необходимости напрямую взаимодействовать с внутренними классами, Reflection и DexKit.

Главная цель проекта — сделать разработку плагинов простой, удобной, стабильной и максимально производительной.

# Цель проекта

ExsteraLib создаётся как единая платформа для разработки плагинов ExsteraGram. Библиотека предоставляет готовые API практически для всех возможностей клиента, позволяя сосредоточиться на реализации логики плагина, а не на работе с внутренними механизмами приложения.
